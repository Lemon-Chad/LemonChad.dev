JPizza

My most significant project is the JPizza programming language, which can be found here. This is a fully functional programming language involving functions, classes, abstraction, and a complete custom bytecode and virtual machine. There is a corresponding package manager and package management system with all packges hosted on my own personal server. There are many expansive libraries, from graphics to servers to even the ability to host websites! This project is rather old, being designed in my sophomore year of high school, and I would love to remake it sometime to show my growth.

Project Ventilium

Project Ventilium, otherwise known as Catacombs, is a high-octane top-down singleplayer shooter game I am currently developing in my free time. Featuring high-speed stylish combat combined with brutal survival mechanics and a punishing gameplay loop, players will be hooked upgrading their weapons and stash and collecting as much gear as possible. Players will attempt to collect as much gold and scrap as they can to purchase, unlock, and upgrade more gear to try and take down the superintelligent AI ruling over the simulation they're trapped in. The GitHub repository for this project is currently private as to withhold assets and development progress. When more publicity is brought, links can be found here.
